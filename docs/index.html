    • Changes in tariffs

    Tariffs and the protection of domestic products have been topic of discussion for a long time and have been back to the spotlight after last election. The topic was brought back by Donald Trump and with renewed media attention some questions arrise, like how does this contribute to differences among countries? We will try to answer this by comparing changes in tariffs among developed and developing countries.

    Using yearly (1996 - 2016) data from the World Bank and World Integrated Trade Solution, we conducted an analysis of the relationship between tariff changes and the economy. The table below sets out the definition of the variables used:

    Dependent Variables Description
    GDP per Capita Growth Rate Year-on-year rate of change of GDP per capita
    Unemployment Rate Unemployed persons as percentage of total labor force
    Labor Force Participation Rate Sum of employed and unemployed persons as a percentage of total population aged 15+
    Inflation Rate Year-on-year increase in consumer prices (%)

    Methodology


    As this is a panel dataset, we took first differences to compute the delta and percentage change. We used delta for GDP per Capita Growth rate changes, and percentage changes for the others, including tariff changes. As the impact of tariff changes are unlikely to be felt immediately, we compared T0 tariff changes to T+2 economic changes.
